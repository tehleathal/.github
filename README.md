# tehleathal/.github

Default GitHub configuration and community health files for repositories under [@tehleathal](https://github.com/tehleathal).

Files here act as fallbacks when a repository does not define its own version. See [GitHub's docs on default community health files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file).

## Contents

| Path | Purpose |
|------|---------|
| `CONTRIBUTING.md` | Default contribution guidelines |
| `CODE_OF_CONDUCT.md` | Default code of conduct |
| `SECURITY.md` | Default security policy |
| `.github/ISSUE_TEMPLATE/` | Default issue templates |
| `.github/PULL_REQUEST_TEMPLATE.md` | Default pull request template |

## Note on visibility

GitHub applies these defaults organization-wide only when this repository is **public**. If this repo is private, use it as a template source and copy files into individual repositories as needed.
